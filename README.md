## 🧾 LedgerOne
Progressive Web Application for personal financial bookkeeping based on a simplified double-entry accounting system.

## ✨ Key Features


## 🛠 Built with
Framework: React 19 + Vite
Language: TypeScript
Database: Supabase
PWA: Vite Plugin PWA + Workbox
